LedgerProposal::changePosition

Inherited from Consensus. Update the position during the consensus process.

Synopsis 
bool
changePosition(
    uint256 const& newPosition,
    NetClock::time_point newCloseTime,
    NetClock::time_point now);
Description

This will increment the proposal's sequence number.

Parameters

Name

Description

newPosition

The new position taken.

newCloseTime

The new close time.

now

the time The new position was taken.

Return Value

true if the position was updated or false if this node has already left this consensus round.

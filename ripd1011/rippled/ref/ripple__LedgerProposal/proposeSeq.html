LedgerProposal::proposeSeq

Inherited from Consensus. Get the sequence number of this proposal.

Synopsis 
std::uint32_t
proposeSeq() const;
Description

Starting with an initial sequence number of seqJoin, successive proposals from a peer will increase the sequence number.

Return Value

the sequence number

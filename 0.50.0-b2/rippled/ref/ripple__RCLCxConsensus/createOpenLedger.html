RCLCxConsensus::createOpenLedger

Create the new open ledger based on the prior closed ledger and any retriable transactions.

Synopsis 
void
createOpenLedger(
    RCLCxLedger const& closedLedger,
    CanonicalTXSet& retriableTxs,
    bool anyDisputes);
Description
Parameters

Name

Description

closedLedger

the ledger just closed that is the starting point for the open ledger

retriableTxs

the set of transactions to attempt to retry in the newly opened ledger

anyDisputes

whether any of the retriableTxs were disputed by us during consensus

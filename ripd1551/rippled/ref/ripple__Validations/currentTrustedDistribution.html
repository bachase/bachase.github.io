Validations::currentTrustedDistribution

Distribution of current trusted validations.

Synopsis 
hash_map< ID, std::uint32_t >
currentTrustedDistribution(
    ID const& currentLedger,
    ID const& priorLedger,
    Seq cutoffBefore);
Description

Calculates the distribution of current validations but allows ledgers one away from the current ledger to count as the current.

Parameters

Name

Description

currentLedger

The identifier of the ledger we believe is current (0 if unknown)

priorLedger

The identifier of our previous current ledger (0 if unknown)

cutoffBefore

Ignore ledgers with sequence number before this

Return Value

Map representing the distribution of ledgerID by count

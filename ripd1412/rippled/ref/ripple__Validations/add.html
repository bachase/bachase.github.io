Validations::add

Add a new validation.

Synopsis 
AddOutcome
add(
    NetClock::time_point t,
    NodeKey const& key,
    Validation const& val);
Description

Attempt to add a new validation.

Parameters

Name

Description

t

Current time

key

The NodeKey to use for the validation

val

The validation to store

Return Value

The outcome of the attempt

Remarks

The provided key may differ from the validation's key() member since we might be storing by master key and the validation might be signed by a temporary or rotating key.

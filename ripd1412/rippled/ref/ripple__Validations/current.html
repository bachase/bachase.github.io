Validations::current

Iterate current validations.

Synopsis 
template<
    class F>
void
current(
    boost::optional< NetClock::time_point > t,
    F&& f);
Description

Iterate current validations, optionally removing any stale validations if a time is specified

Parameters

Name

Description

t

(Optional) Time used to determine staleness

f

Callable with signature (NodeKey const &, |6|Validations const &)

Remarks

This is protected since the implementations/clients primarily iterate trusted validations, but the iteration logic is needed by the currentKeys member function to iterate all validation keys.
